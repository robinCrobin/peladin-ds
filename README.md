# PeladIn

O **PeladIn** é uma plataforma dedicada à marcação de partidas de futebol amador, projetada para simplificar e otimizar o processo de organização de peladas entre amigos e grupos de jogadores.  
Nosso objetivo é fornecer uma solução intuitiva e eficiente que permita aos usuários agendar jogos com facilidade, encontrar o local e horário mais adequados para as partidas e promover a integração social e a diversão através do esporte.

---

## COMPONENTES ESSENCIAIS

1. **Monolito**  
2. **Arquitetura em camadas com MVC**  
3. **REST, com RabbitMQ**  
4. **SQL, caching para consultas**  
5. **Horizontal, com load balancing**  
6. **Autenticação JWT com OAuth2, proteções contra SQL Injection**  
7. **HealthCheck, logs estruturados**
